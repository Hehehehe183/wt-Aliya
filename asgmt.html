TABLE OF CONTENTS

  1. Introduction

    • What is HTML:

      HTML stands for Hypertext markup language.It's a programming language that defines the structure and content of web pages.

    • Basic HTML concepts:

      Basic HTML concepts include HTML tags, elements, comments,tables,and the DOCTYPE declaration.

  2. HTML basics

    • Tags:

      In HTML, tags are used to define the structure and content of a web page.Tags are specified in angle brackets.Here are some common HTML tags:

      • Basic Tags

      html: The root element of an HTML document.
      head: Contains metadata about the document, such as the title, charset, and links to external stylesheets or scripts.
      body: Contains the content of the HTML document.
      title: Specifies the title of the page, which appears in the browser's title bar and in search engine results.
      p: Defines a paragraph of text.

    • Attributes:

      An attribute is a keyword or phrase that provides additional information about an element.They are added to the opening tag of an element and are used to specify various properties.
      example:
      href,src,type,..etc

  3. Advanced HTML

    • Images:

      Images can be placed in html document using img tag.This tag has no closing tag.The most important attribute of img tag is src,which is the location of the image, alt attribute can be used to describe the image

    • Input tags:

      Input tags enables to create many different types of form controls.Form contorls are special tags used in aform to gather information from visitors of a web page.Input tag consisits of an opening tag with attributes.It has no closing tag.label tag is used to specify labels for form feeds
